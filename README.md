## 🎯 Objetivo
Sistema web que permite registrar ventas de una cafetería y analizar los datos mediante reportes y gráficos estadísticos.

## 🏗️ Arquitectura Hexagonal

### Componentes Principales

#### 1️⃣ MANTENEDOR: Productos
- ✅ Crear producto
- ✅ Listar productos
- ✅ Actualizar producto
- ✅ Eliminar producto

#### 2️⃣ PROCESO: Ventas y Análisis
- ✅ Registrar venta
- ✅ Dashboard con gráficos (Chart.js)
- ✅ Filtros por fechas
- ✅ Reportes estadísticos
- ✅ Exportar reportes (PDF/Excel)

## 🛠️ Tecnologías
- **Backend:** Node.js + TypeScript + Express.js + SQLite
- **Frontend:** HTML5 + CSS3 + Bootstrap + JavaScript + Chart.js

## 🚀 Instalación
```bash
npm install
npm run dev
```
